1 First Install Lightbox2 in using CMD
npm install --save lightbox2

Next, open angular.json file and add lightbox.min.css file path to the styles section. Like,

    "styles": [
    "styles.css",
    "node_modules/lightbox2/dist/css/lightbox.min.css"
],

Next, open angular.json add js file

    "scripts": [
    "node_modules/jquery/dist/jquery.js",
    "node_modules/lightbox2/dist/js/lightbox.min.js"
  ],
