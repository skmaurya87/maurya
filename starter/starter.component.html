Start Bootstrap project in angular 7
1 First Install Bootstrap in using CMD
npm install bootstrap --save

Next, open angular.json file and add bootstrap file path to the styles section. Like,

"styles": [
      "node_modules/bootstrap/dist/css/bootstrap.min.css",
      "src/styles.css"
    ],

Refrance bootstrap.min.css angular.json then run ng-build command

2 Install Font awessome in our project if you need
npm install --save font-awesome angular-font-awesome
"styles": [
    "node_modules/bootstrap/dist/css/bootstrap.min.css",
    "node_modules/font-awesome/css/font-awesome.min.css",
    "src/styles.css"
  ],

Refrance font-awesome.min.css angular.json then run ng-build command

3 Install jquery in our project if you need
npm install jquery --save
"scripts": [
    "node_modules/jquery/dist/jquery.js",
  ]

Refrance jquery.min.js file in in angular.json then run ng-build command

Note: you can add multiple jquery file in sequence here example.. 

"scripts": [
      "node_modules/jquery/dist/jquery.js",
      "node_modules/jquery/dist/bootstrap.min.js",
    ]
